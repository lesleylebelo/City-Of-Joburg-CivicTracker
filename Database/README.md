# CivicTrack Database Setup

The civictrack.sql script in the Schema folder will create the `civictrack_db` database along with all required tables, relationships, indexes, and constraints for the CivicTrack system.

## Requirements

Before running the script, ensure the following software is installed:

- SQL Server Express
- SQL Server Management Studio (SSMS)

## Setup Instructions

1. Open SQL Server Management Studio (SSMS)

2. Connect to your SQL Server instance:

localhost\SQLEXPRESS
